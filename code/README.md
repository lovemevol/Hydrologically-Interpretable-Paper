# Public Code Bundle

This directory contains the minimum code bundle for public review of PaperFour.
The files are copied from the project source tree and are provided to document
how the configuration matrix, response dataset, Stage C validation scorecard,
and manuscript figures were generated.

## Contents

`config/`

- `paperfour_param_utils.py`: deterministic fixed-setting constants and Stage
  0/A/B learning-dynamics configuration builders.
- `PaperFour_run_experiments.py`: Stage 0/A/B configuration generator and
  training launcher.
- `PaperFour_run_experiments_eval.py`: Stage C checkpoint-evaluation launcher.
- `reservoir_happo_public_template.py`: redacted public template that summarizes
  the fixed HAPPO setting without exposing restricted workbooks or machine
  settings.

`analysis/`

- `build_response_dataset.py`: builds the Stage B response dataset from
  manifests, learner diagnostics, evaluator exports, and hydrology scorecards.
- `build_stage_c_scorecard.py`: builds the Stage C mixed-sequence validation
  scorecard.
- `build_results_evidence_pack.py`: builds manuscript evidence tables and
  claim-to-source mappings.

## Execution Boundary

The code is sufficient to audit the experimental design and data-processing
logic. Full execution requires files that are not included in the minimum public
package:

- restricted reservoir structure workbook;
- restricted inflow workbooks;
- Stage B trained checkpoints for Stage C evaluation;
- raw learner logs and evaluator operation exports;
- the project Python runtime with DI-engine and reservoir environment modules.

For public review, `config/reservoir_happo_public_template.py` is the preferred
fixed-setting reference. The executable training configuration used in the
private environment is generated by `config/PaperFour_run_experiments.py`.

## Legacy Internal Names

Some source files may retain compatibility-oriented internal names, such as
`stage_b_robust_interval.csv` or checkpoint-related field names. These are
implementation identifiers and should be interpreted according to the manuscript
terms: sampled acceptable values, representative checkpoints, and fixed
observation setting.
