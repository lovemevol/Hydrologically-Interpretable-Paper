# Minimal Reproducible Results Plan

## Reproducibility Target

The minimum public reproducibility target is configuration-level auditability.
An external reviewer should be able to verify:

1. what was fixed across all experiments;
2. which six learning-dynamics parameters were varied;
3. which configurations were included in Stage A screening and Stage B repeated
   sensitivity analysis;
4. how Stage B seeds were assigned;
5. how representative checkpoints were selected for Stage C;
6. which response-field groups and metric directions were used in the reported
   analysis.

## Optional Result-Level Materials

If data-use constraints allow, the following derived files are recommended for a
stronger reproducible-results submission:

- Stage B response dataset or a redacted aggregate version;
- Stage B configuration summary;
- Stage B parameter-correlation and interaction tables;
- Stage C scorecard;
- figure-source CSV files generated by the PaperFour figure scripts.

Raw reservoir workbooks, raw inflow workbooks, trained checkpoints, and raw
daily operation workbooks are not part of the minimum public package unless the
data owner permits release.

## Reviewer-Facing Boundary

The reproducible-results package supports the manuscript's fixed-setting
sensitivity claim. It does not provide enough evidence to claim a dense global
response surface, transferable HAPPO parameters, cross-basin transferability, or
broad multi-scenario hydrologic robustness.
