# Data, Model, and Code Availability Statement

A minimum public reproducibility package is available in the public GitHub
repository `https://github.com/lovemevol/Hydrologically-Interpreted-Paper`,
under `public_reproducibility_package/`. The released materials include the
fixed-control settings, HAPPO learning-dynamics parameter space, Stage 0,
Stage A, and Stage B configuration matrices, Stage C mixed-sequence validation
metadata, representative-role mapping, seed and checkpoint-selection protocol,
response-dataset schema, public code bundle, and figure-generation scripts.

The public repository does not include restricted reservoir-system workbooks,
raw inflow workbooks, trained checkpoints, raw evaluator workbooks, or large
training logs. Those files contain nonpublic operational inputs or are large
derived artifacts. The public materials are intended to support auditability of
the fixed experimental boundary and sampled learning-dynamics matrix within
the fixed five-reservoir HAPPO setting studied in the manuscript.
