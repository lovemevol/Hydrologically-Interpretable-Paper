# Data, Model, and Code Availability Statement

The data, models, and code that support the findings of this study are
available in the public GitHub repository at
`https://github.com/lovemevol/Hydrologically-Interpreted-Paper`, under
`public_reproducibility_package/`. The package includes the fixed-control
settings, HAPPO learning-dynamics parameter space, Stage 0, Stage A, and Stage
B configuration matrices, Stage C mixed-sequence validation metadata,
representative-role mapping, seed and checkpoint-selection protocol,
response-dataset schema, and public code bundle. Restricted reservoir-system
workbooks, inflow workbooks, trained checkpoints, raw evaluator workbooks, and
large training logs are not publicly released.
