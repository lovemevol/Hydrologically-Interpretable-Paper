# Data, Model, and Code Availability Statement

A minimum public reproducibility package will be provided upon publication.
The released materials will include the fixed-control settings, HAPPO
learning-dynamics parameter space, Stage 0, Stage A, and Stage B configuration
matrices, Stage C mixed-sequence validation metadata, representative-role
mapping, seed and checkpoint-selection protocol, response-dataset schema, and
public code bundle.

The public package will not include restricted reservoir-system workbooks, raw
inflow workbooks, trained checkpoints, raw evaluator workbooks, or large
training logs. Those files contain nonpublic operational inputs or are large
derived artifacts. The public materials are intended to support auditability
of the fixed experimental boundary and sampled learning-dynamics matrix within
the fixed five-reservoir HAPPO setting studied in the manuscript.
