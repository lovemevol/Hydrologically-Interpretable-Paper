# Data, Model, and Code Availability Statement

The public materials supporting this manuscript are available at
\url{https://github.com/lovemevol/Hydrologically-Interpretable-Paper},
including the training parameter configurations, random seeds, checkpoint
selection protocol, Stage B/Stage C configuration matrices, mixed-sequence
validation metadata, representative-role mapping, response-dataset schema,
and public configuration and analysis scripts.

The historical inflow and operational workbooks, trained checkpoints, and raw
evaluator workbooks are subject to institutional access and operational-security
restrictions and are not included in the public repository.

The released configuration materials are intended to reproduce the sampled
training-parameter matrix and audit the reported fixed experimental boundary.
They are not intended to imply a transferable parameter prescription beyond the
fixed five-reservoir HAPPO setting studied in the manuscript.
